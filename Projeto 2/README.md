# Second View of analysis of Data Jobs available globally in 2024


### Introduction
A slightly different approach to the visualization of Data Jobs Postings. In this new version it is possible to filter by Country and Job Title and a deeper dive into skills and the amount of jobs requiring each skill.

![Dashboard](/images/Project2_Dashboard.png)

You can find the file for the dashboard here: [`Project2_Dashboard.pbix`](Project2_Dashboard.pbix).

-   **Dax Fundamentals** Creating calculations and aggregations to derive key insights.
